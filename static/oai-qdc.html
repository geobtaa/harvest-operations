OAI QDC Harvester

Purpose

This harvester processes qualified Dublin Core OAI-PMH records from local XML snapshots and writes a primary CSV plus a distributions CSV.

Before You Run It

Download the source XML separately in the terminal with scripts/oai_download.py before using this page. This HTML view only runs harvesters/oai_qdc.py against files that already exist locally.

Currently supported sources in this view are University of Iowa Libraries and University of Washington Libraries.

Required input
Downloaded XML must already be present in the configured local OAI download folder for the selected source.
