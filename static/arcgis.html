ArcGIS Harvester

Purpose

This script scans the DCAT 1.1 APIs of ArcGIS Hubs and returns metadata for all suitable items. It produces two CSV files:

  • A Primary CSV containing all metadata except the link fields
  • A Distribution CSV with only the link fields (e.g., landing page, download, REST services)

Inputs

The ArcGIS harvester now uses the workflow inputs generated by the harvest task dashboard together with ArcGIS Hub metadata defaults stored in reference_data/arcHub_metadata.csv.

You do not need to manually download or upload source CSVs from this page before running the harvester.
