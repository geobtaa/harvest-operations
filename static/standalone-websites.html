Standalone Website Link Check

Checks the landing-page URL in the Identifier column for every standalone website (w00_01) in reference_data/websites.csv. Restricted records are recorded as skipped and are never requested. A live log identifies each site as it is checked.

Outputs

A dated metadata CSV is written to outputs/YYYY-MM-DD_standalone-websites_primary.csv. It contains only standalone website rows. Successful website rows receive today’s Date Accessioned; failed checks are moved to draft with an Admin Note. The standalone-websites harvest record is appended last.
