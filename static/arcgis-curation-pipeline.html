This is the staged arcgis_curation_pipeline for selected ArcGIS Hub datasets. Choose a YAML job, harvest its metadata, edit the generated CSV manually, confirm that review, and then run the remaining tasks separately or together.

Jobs come from curation/jobs/*.yaml. The fill-in template is not runnable and does not appear in the list.
