Transform OGM Aardvark Metadata

This job reads nested JSON metadata files from inputs/ogm_aardvark, maps matching fields directly to the GeoBTAA schema, preserves non-schema source fields in the primary CSV, and writes a separate distribution CSV from dct_references_s.

Outputs:

  • outputs/YYYY-MM-DD_ogm_aardvark_primary.csv
  • outputs/YYYY-MM-DD_ogm_aardvark_distributions.csv
