Harvest Task Dashboard

This job reads inputs/harvest-records.csv and reference_data/websites.csv, calculates due dates from Last Harvested and Accrual Periodicity, enriches tasks with matching website metadata, and writes per-workflow harvest input CSVs.

Use Generate Dashboard Files & Prepare Inputs after replacing the source CSVs. This writes the report outputs into reports/ and prepares inputs/harvest-workflow-inputs/ by splitting harvest-records.csv by Harvest Workflow.

The workflow queue below identifies harvest jobs that are due now or within the next month and links to the matching static harvester page when one exists.
