Harvest Task Dashboard

This job reads inputs/harvest-records.csv and inputs/websites.csv, calculates due dates from Last Harvested and Accrual Periodicity, enriches tasks with matching website metadata, and writes per-workflow website input CSVs.

Use Generate Dashboard Files to write the CSV and HTML outputs into reports/. Workflow input splits are written to inputs/. The live preview below renders the current inputs directly from FastAPI.

Each task row in the preview includes an issue button that opens a prefilled GitHub issue using the harvest-task template for the configured target repository.

Get Latest Source CSVs

Download the newest files before running the dashboard, then save them into inputs/ with these names:

  • harvest-records.csv -> save as inputs/harvest-records.csv
  • websites.csv -> save as inputs/websites.csv
