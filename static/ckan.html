CKAN Harvester

Purpose

This script scans a CKAN package search API and returns dataset metadata. It produces two CSV files:

  • A Primary CSV containing all metadata except the link fields
  • A Distribution CSV with only the link fields (e.g., landing page, downloads, web services)

Inputs

The CKAN harvester uses settings from config/ckan.yaml, including the base URL, package search endpoint, and metadata defaults.

Outputs

The harvester writes dated output files under outputs/:

  • outputs/YYYY-MM-DD_ckan_primary.csv: harvested dataset records.
  • outputs/YYYY-MM-DD_ckan_distributions.csv: distribution links derived from the primary output.
  • reports/ckan/YYYY-MM-DD_ckan_report.csv: per-site run status, record counts, and totals.
